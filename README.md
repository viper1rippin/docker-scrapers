# Universal Scraper Sandbox

Multi-runtime Docker environment for web scraping supporting Node.js, Python, Deno, Go, Ruby, and browser automation tools.

## Architecture

- **Browser Automation Containers**: Playwright, Puppeteer, Selenium
- **Universal Runtime Sandbox**: Execute scrapers from GitHub repos in multiple languages
- **Nginx Load Balancer**: Rate limiting, routing, and SSL termination

## Quick Start

See docker-compose.yml for configuration details.
